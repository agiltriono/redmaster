B) Installation

Installing the rating requires the following steps.

  1. Including the rating's css file.
  2. Specifying the html tags of the rating within your page.

Below describes these steps in more detail.

In step 1, to include css files, simply specify the following lines of code within your <head> tag: 

<link rel="stylesheet" type="text/css" href="css/css-star-rating.css" />
<link rel="stylesheet" type="text/css" href="http://maxcdn.bootstrapcdn.com/font-awesome/4.6.3/css/font-awesome.min.css" />

In step 2, to specify the tags of the rating itself, refer to the examples in the deploy/index.html file.

C) Colors

Rating widget comes with 6 predefined colors:

orange red green blue purple black

You can add colors to the rating like that:

<div class="css-star-rating red">

D) Sizes

Rating widget comes with 3 sizes:

small medium large

You can add sizes to the rating like that:

<div class="css-star-rating medium">

You can combine colors and sizes by adding them to the HTML code.

<div class="css-star-rating large blue">

E) Working with PHP

Whether you are including the rating widget in an exsting form or a new form all you need to do to post the rating value is make sure that each rating widget has a name attribute. For example a services rating widget will have a name attribute as name="services" for all radio inputs we shall use that name to post the value e.g value="4.5" in php.

Lets assume you have a review form that has an action as formprocess.php like this below:

<form method="post" action="php/formprocess.php" id="review">

The processing script in the above form is formprocess.php where you will put your PHP code for handling our services rating widget below.
