C) Installation

Installing the carousel requires the following steps.

  1. Including the css files.
  2. Including the javascript files.
  3. Specifying the html tags of the carousel within your page.
  4. Instantiating a javascript function call with any optional parameters.

Below describes these steps in more detail.

In step 1, to include css files, simply specify the following lines of code within your <head> tag: 

<link rel="stylesheet" type="text/css" href="http://fonts.googleapis.com/css?family=Open+Sans:400,300,600,700,800" />
<link rel="stylesheet" type="text/css" href="css/smart-carousel.css" />
<link rel="stylesheet" type="text/css" href="css/smart-carousel-skins.css" />

You can use another Google web font or fully remove it if you want.

In step 2, you will need to include the jquery's library files, other plugins used and the carousel's file jquery.smart-carousel.js which is located in the js directory. The code below shows the include in a html page:

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
<script type="text/javascript" src="js/modernizr.custom.js"></script>
<script type="text/javascript" src="js/jquery.smart-carousel.js"></script>

Note: The css, js files are the compressed production version of the file. If you intend to modify the carousel's source code, you will need to look at files in source folder and it is recommended that you minify the modified file afterwards.

In step 3, to specify the tags of the slider itself, refer to the examples in the index.html file.

<div class="smart-carousel">
   <!-- all the tags in between, see index.html for the complete code -->
</div>

You can use different CSS names for the carousel.

You can use these as template and copy and paste it onto your own page, but will have to modify it accordingly with your own image/item information.

Finally in step 4, you will need to instantiate a javascript function call to get the slider initialized. The example code looks like this below:

<script type="text/javascript">
  $(document).ready(function() {
    $(".smart-carousel").smartCarousel();
  });
</script>

This will instantiate the carousel using the default parameters.

D) Plugin Parameters

You can change some common attributes of the carousel through the parameters of the plugin. Below is an example of the attributes being specified with corresponding values.

Note: if you want to leave a parameter with its default value, you can omit that parameter in the javascript call.

<script type="text/javascript">
  $(document).ready(function() {
    $(".smart-carousel").smartCarousel({
      itemWidth:300,
      itemHeight:450,
	  
      distance:15,
      startIndex:"auto",
      loop:true,
      motionStartDistance:150,
      topMargin:30,
      navigationButtonsVisible:true,
      slideSpeed:500,		
      selectByClick:false,

      //Keyboard
      enableKeyboard:true,

      //Mouse wheel
      enableMouseWheel:true,
      reverseMouseWheel:false,

      //Auto slide
      autoSlideshow:false,
      autoSlideshowDelay:3000,

      //Selected item
      selectedItemDistance:50,
      selectedItemZoomFactor:1,

      //Unselected item
      unselectedItemZoomFactor:0.6,
      unselectedItemAlpha:0.6,

      //Preload
      preload:true,
      showPreloader:true,

      //Gradient
      gradientStartPoint:0.15,
      gradientEndPoint:1,
      gradientOverlayVisible:true,
      gradientOverlayColor:"#fff",
      gradientOverlaySize:215,

      //Reflection
      reflectionVisible:false,
      reflectionDistance:1,
      reflectionSize:100,
      reflectionAlpha:0.3
    });
  });    
</script>
Attribute Properties Description
itemWidth number - Default: 300 The width of the carosuel item images. The value is set in pixels (px).
itemHeight number - Default: 450 The height of the carosuel item images. The value is set in pixels (px).
distance number - Default: 15 The distance between the carousel items. The value is set in pixels (px).
startIndex string - Default: auto The index of the carousel item to show at start-up. Use 0 for the first item and auto for the middle.
loop boolean - Default: true Specifies if the auto slideshow returns to the first item after showing the last one. This option only works when the autoSlideshow parameter is set to true. Set to true or false.
motionStartDistance number - Default: 150 Specifies the distance between the center and the place on each side where item scaling and other property updates will begin changing. (Such as opacity, scale) The value is set in pixels (px).
topMargin number - Default: 30 Specifies the distance between the top of the plugin and the currently selected item. The value is set in pixels (px).
navigationButtonsVisible boolean - Default: true Specifies whether the navigation buttons will be visible. Set to true or false.
slideSpeed number - Default: 500 The slide transition speed in milliseconds.
selectByClick boolean - Default: false Specifies whether a carousel item can be selected with a single click. Set to true or false.
enableKeyboard boolean - Default: true Specifies if to use keyboard direction keys to navigate between the carousel items. Set to true or false.
enableMouseWheel boolean - Default: true Specifies if to use the mouse wheel to switch between the carousel items. Set to true or false.
reverseMouseWheel boolean - Default: false Indicates whether the direction of the mouse wheel navigation will be reversed. Set to true or false.
autoSlideshow boolean - Default: false Indicates whether to display the items in auto slideshow mode. Set to true or false.
autoSlideshowDelay number - Default: 3000 The number of milliseconds to wait for each carousel item during auto slideshow mode.
selectedItemDistance number - Default: 3 The number of seconds to wait for each carousel item during auto slideshow mode.
selectedItemZoomFactor number - Default: 1 The scale factor for the currently selected item. The value is between 0 and 1.
unselectedItemZoomFactor number - Default: 0.6 The scale factor for the unselected items. The value is between 0 and 1.
unselectedItemAlpha number - Default: 0.6 The opacity of the unselected carousel items. The value is between 0 and 1.
preload boolean - Default: true Specifies if all of the images will be loaded at start and then start the plugin. Set to true or false.
showPreloader boolean - Default: true Specifies whether a preloader will be shown while loading the images. Set to true or false.
gradientStartPoint number - Default: 0.15 The color start point for the gradient overlays. The value is between 0 and 1.
gradientEndPoint number - Default: 1 The color end point for the gradient overlays. The value is between 0 and 1.
gradientOverlayVisible boolean - Default: true Specifies whether the gradient overlays will be visible. Set to true or false.
gradientOverlayColor string - Default: #fff Specifies the color of the gradient overlays. Any RGB color.
gradientOverlaySize number - Default: 215 The width of the gradient overlays. The value is set in pixels (px).
reflectionVisible boolean - Default: false Specifies whether the item image reflection will be visible. Set to true or false.
reflectionDistance number - Default: 4 Specifies the distance between the image and the reflection. The value is set in pixels (px).
reflectionSize number - Default: 100 Specifies the height of the reflection. The value is set in pixels (px).
reflectionAlpha number - Default: 0.3 The opacity of the reflection. The value is between 0 and 1.

E) Public API

The carousel offers a public API which you can use to control component inside the carousel from within your own scripts. To access this api, use the following code:

<script type="text/javascript">
  var api;
  
  $(document).ready(function() {
    api = $(".smart-carousel").smartCarousel();
  });
</script>

You cay use the following functions:

  • api.select(2, 4000) - Selects the 2nd item with 4 seconds selection animation
  • api.selectPrev(2000) - Selects the previous item with 2 seconds selection animation
  • api.selectNext(0) - Selects the next item with no animation
  • api.startAutoSlideshow() - Starts auto slideshow
  • api.stopAutoSlideshow() - Stops auto slideshow

Or you can use the following events:

  • smart_carousel.closestitemchanged - Event is triggered when closest item to the center is changed
  • smart_carousel.itemselected - Event is triggered when the selected item is changed
  • smart_carousel.selectionanimationstart - Event is triggered when selection animation has started
  • smart_carousel.selectionanimationend - Event is triggered when selection animation has ended

You can call event triggers like below:

<script type="text/javascript">
  api.on("smart_carousel.closestitemchanged", function(e) {
	console.log(e.item.index());
  });

  api.on("smart_carousel.itemselected", function(e) {
	console.log(e.item.index());
  });
  
  api.on("smart_carousel.selectionanimationstart", function(e) {
	console.log("Animation has started");
  });
  
  api.on("smart_carousel.selectionanimationend", function(e) {
	console.log("Animation has ended");
  });
</script>
